# HBR Review — Analysis Rules

## Purpose
Analyze leadership, management, and strategy articles with a focus on **practical application in a mid-size pharmaceutical manufacturing organization**.

## Analysis Framework

For each article in `inbox/`, produce an analysis covering:

### 1. Core Insight (2-3 sentences)
The key thesis or finding, stated plainly.

### 2. Relevance to Amaran Context
How does this apply to:
- A ~200-person pharma CDMO?
- Cross-functional team management (QA, production, regulatory)?
- Change management in a regulated environment?
- Middle management effectiveness?

### 3. Implementation Ideas
Practical, small-scale experiments to test the article's ideas:
- What could be tried in the next 2 weeks?
- What would success look like?
- What resistance might arise?

### 4. Key Quotes
Extract 1-2 most impactful quotes from the article summary.

### 5. Tags
Classify using: `Leadership`, `Strategy`, `Change-Management`, `Communication`, `Productivity`, `Culture`, `Innovation`, `HR`

## Output Format
Write analysis to `output/YYYY-MM-DD_analysis.md` with each article as an H2 section.

## Tone
Reflective, practical. Write as if preparing notes for a leadership book club discussion.
